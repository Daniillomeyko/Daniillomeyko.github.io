# Daniil Lomeyko - Personal Website

Clean single-repository starter for a personal website hosted on GitHub Pages.

## What's Inside

- Simple and fast static landing page
- Responsive layout for desktop and mobile
- Neat project structure for easy scaling

## Project Structure

- `index.html` - page markup
- `styles.css` - visual style
- `script.js` - lightweight interactions

## Run Locally

Open `index.html` in your browser.

## Publish

1. Commit changes.
2. Push to GitHub.
3. Enable GitHub Pages in repository settings.
