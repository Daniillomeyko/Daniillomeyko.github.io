# Старт для новичка (5-10 минут)

Этот файл поможет быстро запустить и менять сайт без страха "сломать все".

## 1) Что установить на macOS

Сначала один раз установи системные инструменты:

```bash
xcode-select --install
```

Потом проверь Python:

```bash
python3 --version
```

## 2) Первый запуск проекта

В папке проекта `lift_site` выполни:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```

Сайт откроется на `http://127.0.0.1:5000`.

## 3) Какие файлы менять

- Главная: `templates/home.html`
- Услуги: `templates/services.html`
- Контакты: `templates/contacts.html`
- Общий шаблон (шапка/футер): `templates/base.html`
- Доп. стили главной: `static/css/home.css`
- Общие данные компании: `.env` (`COMPANY_ADDRESS`, `COMPANY_CITY`)
- Реквизиты и сервисные данные: `.env` (`COMPANY_INN`, `COMPANY_OGRN`, `COMPANY_WORK_HOURS`)

## 4) Что можно менять безопасно

- Тексты в блоках
- Заголовки кнопок
- Контактные данные
- Цвета и отступы в CSS

Если меняешь только это, обычно ничего не ломается.

## 5) Быстрый чек после изменений

Проверь страницы:

- `/`
- `/services`
- `/contacts`
- `/admin`

И отдельно с телефона (или devtools mobile view), чтобы верстка не "поехала".

## 6) Если сайт не запускается

- Убедись, что активирована виртуальная среда:
  `source .venv/bin/activate`
- Проверь, что есть `.env`
- Установи зависимости заново:
  `pip install -r requirements.txt`
