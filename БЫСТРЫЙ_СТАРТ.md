# 🚀 БЫСТРЫЙ ЗАПУСК САЙТА ООО "ЭТАЖИ"

## 📦 1. КОПИРОВАНИЕ НА СЕРВЕР

```bash
scp lift_site_deploy.zip root@ваш-ip:/var/www/
```

---

## ⚙️ 2. УСТАНОВКА (5 МИНУТ)

```bash
cd /var/www && unzip -o lift_site_deploy.zip -d lift_site && cd lift_site

python3 -m venv venv && source venv/bin/activate

pip install --upgrade pip && pip install -r requirements.txt && pip install gunicorn
```

---

## 🔐 3. ПРОВЕРКА .ENV

```bash
nano .env
```

**Проверьте:**
```env
MAIL_PASSWORD=ваш_пароль_приложения
FLASK_DEBUG=False
```

---

## ▶️ 4. ЗАПУСК

```bash
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

**Проверка:** http://ваш-ip:5000

---

## 🌐 5. NGINX + SSL (10 МИНУТ)

```bash
sudo apt update && sudo apt install nginx certbot python3-certbot-nginx -y

sudo nano /etc/nginx/sites-available/lift_site
```

**Вставьте:**
```nginx
server {
    listen 80;
    server_name _;
    location / { proxy_pass http://127.0.0.1:5000; proxy_set_header Host $host; proxy_set_header X-Real-IP $remote_addr; }
}
```

```bash
sudo ln -s /etc/nginx/sites-available/lift_site /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl restart nginx

sudo certbot --nginx -d etazhi-lifts.online -d www.etazhi-lifts.online
```

---

## 🔧 6. АВТОЗАПУСК

```bash
sudo nano /etc/systemd/system/lift_site.service
```

**Вставьте:**
```ini
[Unit]
Description=ООО ЭТАЖИ
After=network.target

[Service]
User=root
WorkingDirectory=/var/www/lift_site
Environment="PATH=/var/www/lift_site/venv/bin"
ExecStart=/var/www/lift_site/venv/bin/gunicorn -w 4 -b 127.0.0.1:5000 app:app
Restart=always

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload && sudo systemctl start lift_site && sudo systemctl enable lift_site
```
